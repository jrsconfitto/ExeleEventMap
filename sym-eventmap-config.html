About

Event Map symbol for visualizing Event Frames. By default the cell size is proportional to the Event Frame duration and the cell color indicates the Event Frame template. If a template filter is applied then attributes can be selected to control the size and color of the cell.

Hovering over a cell will show summary data about the Event Frame. Clicking on a cell displays attribute values in a table.

This symbol was created by Exele, a PI partner specializing in PI products, services, and PI based environmental monitoring.
