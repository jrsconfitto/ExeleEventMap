About

Treemap symbol for visualizing Event Frames. By default the size of the displayed checkbox is proportional to the Event Frame duration. The color of the box indicates the Event Frame template. The view can be scoped to a single template, then the size and color of the box can be based on attributes.

Hovering over a box will show summary data about the Event Frame. Clicking on a box will attribute values in the table. Total time off all displayed Event Frames is shown below the Treemap.

This symbol was created by Exele, a PI partner specializing in PI products, services, and PI based environmental monitoring.
